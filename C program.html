Understanding Functions in C

a. Define Function

A function in C is a modular block of code used to carry out a specific operation. Functions enhance code reusability and simplify program logic by dividing tasks into separate, manageable sections.

Example: A function that multiplies two numbers and returns the product:

int multiply(int x, int y) {
  return x * y;
}

b. Syntax of Functions

The basic structure of a function includes a return type, function name, parameters (if any), and a code block enclosed in curly braces.

return_type function_name(parameter_list) {
  // statements
}

c. Types of Functions

In C, functions are mainly categorized as:

  • Library Functions – Predefined by C libraries, such as puts(), getchar(), or sqrt().
  • User-defined Functions – Created by the programmer to perform specific tasks that are not covered by standard libraries.

Example of a user-defined function:

void showMessage() {
  puts("Welcome to Programming in C!");
}

d. Components of Function

Each function in C is made up of the following parts:

i. Function Prototype

This line informs the compiler about the function's signature before its actual definition.

float divide(float, float);

ii. Function Call

Triggers the function's execution by specifying its name with actual arguments.

float result = divide(10.0, 2.5);

iii. Function Definition

This contains the full code of the function.

float divide(float a, float b) {
  return a / b;
}

iv. Return Type

The return type specifies what kind of value the function sends back. For example:

char getGrade();
int computeSum();
void printLine();

Function Classification in C

1. Function with No Parameters and Value-Returning

This function doesn't take input but returns a value. Example: returns a fixed discount rate.

float getDiscount() {
  return 5.0;
}

2. Function with Parameters and Value-Returning

Takes arguments and returns the computed result. Example: Find the greater of two numbers.

int max(int x, int y) {
  return (x > y) ? x : y;
}

3. Function with No Parameters and Non-Returning

Performs a task without accepting input or returning a value. Example: displays a greeting message.

void welcomeMessage() {
  printf("Good day!\n");
}

4. Function with Parameters and Non-Returning

Uses arguments to perform an action but doesn't return any result. Example: print stars based on input number.

void printStars(int n) {
  for (int i = 0; i < n; i++) {
    printf("*");
  }
}

Understanding Storage Classes in C

i. Automatic Storage Class

void check() {
  int flag = 1; // auto by default
  if (flag)
    printf("Active\n");
}

ii. External Storage Class

extern int total; // shared across files

iii. Register Storage Class

void loopFast() {
  register int i;
  for (i = 0; i < 100; i++) {
    // do something
  }
}

iv. Static Storage Class

void counter() {
  static int count = 0;
  count++;
  printf("%d\n", count);
}

Understanding Recursive Functions in C

Example: Sum of Natural Numbers Using Recursion

#include <stdio.h>

int sum(int n);

int main() {
  int num = 5;
  printf("Sum = %d", sum(num));
  return 0;
}

int sum(int n) {
  if (n == 1) return 1;
  return n + sum(n - 1);
}

Demonstration of Passing Array to a Function in C

Example 1: Calculating Average

#include <stdio.h>

float average(int arr[], int size) {
  int sum = 0;
  for (int i = 0; i < size; i++) {
    sum += arr[i];
  }
  return (float)sum / size;
}

int main() {
  int marks[] = {70, 80, 90};
  printf("Average: %.2f\n", average(marks, 3));
  return 0;
}

Example 2: Inverting Array Signs

#include <stdio.h>

void invertSigns(int nums[], int len) {
  for (int i = 0; i < len; i++) {
    nums[i] = -nums[i];
  }
}

int main() {
  int values[] = {5, -10, 15};
  invertSigns(values, 3);
  for (int i = 0; i < 3; i++) {
    printf("%d ", values[i]);
  }
  return 0;
}
